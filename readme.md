🚀 AI Against Misinformation

A modern web platform built using Next.js to detect, analyze, and combat misinformation such as fake news, deepfakes, and AI-generated content.

📌 Project Overview

With the rise of AI-generated content, misinformation is spreading rapidly across digital platforms. This project aims to:

Detect fake news and misleading content
Analyze text, images, and videos
Provide real-time verification
Support multiple languages (especially for India 🇮🇳)
🛠️ Tech Stack
Frontend: Next.js 16, React 19
Styling: Tailwind CSS
Icons: Lucide React
Language: TypeScript
Linting: ESLint
